# Alauda Global Recruitment OS (GROS) Copilot - 用户操作手册

## 核心功能操作指南

### 📋 模块零：HC 业务需求审批 (业务线 & HR 协同起点)
**使用场景**：业务主管需要招人，HR 需要审核需求合理性。告别口头沟通，实现需求线上化流转。

1. **业务线主管 (提报)**：
   - 进入模块零的 `[📤 我是业务：提报新 HC]` 页面。
   - 用大白话填入要解决的核心麻烦（Mission）、红线（Deal Breakers）等信息，点击提交。
2. **HRBP (审批)**：
   - 切换到 `[✅ 我是 HR：审批 HC 需求]` 页面。
   - 查看业务线提交的工单，点击蓝色的“批准”或红色的“驳回”。批准后的 HC 会自动流入下一个模块。

### 🎯 模块一：JD 逆向与自动寻源引擎
**痛点解决**：拒绝“输入垃圾，输出垃圾”。一键生成顶级英文 JD 与极客级猎头搜索词。

1. **承接业务需求**：在页面最上方的下拉菜单中，选择刚才在“模块零”中**已经批准的 HC**。系统会瞬间将业务主管的大白话需求填充到下方表单中。
2. **专业润色与生成**：HR 可微调参数后，点击“运行系统”，获取高转化率的 JD 与 Google X-Ray 自动化寻源代码（可以直接复制到浏览器搜索 GitHub / LinkedIn 人才）。

### ✉️ 模块二：自动化触达 (Cold Outreach)
**痛点解决**：抛弃废话模板，用高度个性化的营销级话术直击候选人痛点，大幅提升高端人才回复率。

1. **输入候选人情报**：系统自动读取模块一生成的职位信息，你只需补充候选人的名字和在领英上的核心亮点（如：曾在竞品公司工作过）。
2. **生成破冰话术**：一键生成结构严谨的长篇邮件（Email版）和短小精悍的私信（InMail版），均采用地道高级商务英语。

### 📝 模块三：结构化面试评估系统 
**痛点解决**：统一全球各部门面试官的评分标准，消除面试随机性和个人偏见。

1. **生成评估卡**：承接前序生成的 JD，系统会自动拆解出“技术胜任力、售前交付力、文化契合度”三个核心维度。
2. **分发给考官**：系统会输出一份带 1-5 分严格定义标准的【行为锚定评分卡 (BARS)】以及对应的 STAR 深度追问题库。建议在面试前下载 Markdown 发送给所有考官。

### 📚 模块四与五：知识库自生长与 RAG 智库 
**痛点解决**：打破出海政策“黑盒”，解决初创出海团队没有现成大部头知识库的冷启动问题。

* **模块五（录入）**：HR 可随时将零散的踩坑经验（如某国工作签的潜规则）用大白话录入系统。点击“编译”，系统会自动排版生成企业专属的动态 Playbook。
* **模块四（问答）**：在这里向 AI 提出出海相关的政策、薪酬问题。AI 会基于官方静态 PDF 与大家共建的动态 Playbook 进行检索回答，并提供精准的溯源。
